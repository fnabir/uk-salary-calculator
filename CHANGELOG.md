# Changelog

## [1.0.0] - 2026-06-30

### Added

- Full tax configuration for the 2026/27 tax year (UK + Scotland)
- Complete PAYE calculation engine written in pure TypeScript
- Support for personal allowance tapering, salary sacrifice, and relief-at-source pensions
- National Insurance, student loan, and tax band breakdown calculations
- Annual, monthly, and weekly calculation modes
- Detailed tax band breakdown output
- Utility helpers for formatting, transitions, and UI behaviour
- Main UI built with Next.js 16 + TypeScript 5
- Smooth animations and responsive layout
- Initial deployment of the project
